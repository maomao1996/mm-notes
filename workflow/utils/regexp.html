344 格式手机号 ​

从左到右匹配

js
/(^\d{3}|\d{4}\B)/g

/* 🌰 */
'15512341234'.replace(/(^\d{3}|\d{4}\B)/g, '$1 ')

从右到左匹配

js
/\B(?=(\d{4})+\b)/g

/* 🌰 */
'15512341234'.replace(/\B(?=(\d{4})+\b)/g, ' ')
